# Flutter Admin UI Best Practices

## Design Guidelines
- Sử dụng layout rộng (Bottom Nav hoặc Sidebar)
- DataTable / Danh sách có phân trang
- Card thống kê cho Dashboard
- Form quản lý rõ ràng + validation
- Status chips (Pending, Approved, Shipped...)
- Image picker + preview

**Luôn kết hợp** với skill `flutter-ui-consistency` khi tạo màn mới.
